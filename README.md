# picobot-brain\n\nGo-native knowledge brain for AI agents — hybrid RAG search, self-wiring knowledge graph, zero dependencies.\n\nDesigned for [Picobot](https://github.com/WLTBAgent/picobot) but usable by any Go project.\n\n## Features\n\n- **Hybrid Search** — FTS5 (BM25) + vector similarity, merged via Reciprocal Rank Fusion\n- **Knowledge Graph** — auto-extracted entities and typed edges from [[wikilinks]], @mentions, and relationship patterns\n- **SQLite-native** — pure Go via modernc.org/sqlite, no CGO, runs on Raspberry Pi\n- **Embeddings** — local (Ollama/nomic-embed-text) or remote (OpenAI-compatible) APIs\n- **Content dedup** — SHA-256 content hashing prevents duplicate ingestion\n- **Graceful degradation** — FTS5-only mode when no embedding provider is available\n\n## Quick Start\n\n```go\nimport brain "github.com/WLTBAgent/picobot-brain"\n\n// Init with local Ollama embeddings\nembedder := brain.NewOllamaProvider(brain.OllamaConfig{Model: "nomic-embed-text"})\nb, _ := brain.Init("brain.db", embedder, brain.DefaultOptions())\n\n// Ingest pages\nb.IngestPage(ctx, brain.Page{\n    Slug: "notes/hello", Title: "Hello",\n    Content: "Notes about Go and Raspberry Pi",\n})\n\n// Import existing markdown directory\nb.IngestDir(ctx, "notes", "/path/to/notes/")\n\n// Search\nresults, _ := b.Search(ctx, "Raspberry Pi Go", brain.SearchOpts{Limit: 5})\n\n// Maintain (backfill embeddings, extract entities, prune orphans)\nb.Maintain(ctx)\n```\n\n## No Embeddings? No Problem\n\n```go\n// FTS5-only mode — still better than keyword matching alone\nb, _ := brain.Init("brain.db", nil, brain.DefaultOptions())\n```\n\n## Embedding Providers\n\n| Provider | Model | Size | Quality (MTEB) | Runs on Pi? |\n|----------|-------|------|----------------|-------------|\n| Ollama local | nomic-embed-text | 274MB | 62.39 | ✅ Pi 5 |\n| Ollama local | granite-embedding:30m | 60MB | ~50 | ✅ Pi Zero 2W |\n| Remote API | text-embedding-3-small | — | 62.3 | N/A |\n\n## License\n\nMIT
